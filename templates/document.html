1. Libreria ReportBro Designer

ReportBro Designer es un plugin JavaScript que se puede integrar en su aplicación web. Aunque esto requiere un poco más de trabajo al principio, tener una herramienta de diseño de plantillas integrada tiene bastantes beneficios como descubrirás leyendo. La instalación es sencilla incluyendo algunas librerías JavaScript e inicializando el Diseñador con algo de código HTML y JavaScript.

Está optimizado para informes sencillos, es decir, para mostrar datos (texto, datos tabulares o imágenes) exactamente como se definen en una plantilla de diseño. Todos los parámetros se pasan directamente al informe y, por lo tanto, ReportBro no requiere conocimientos de SQL (u otro lenguaje de consulta) para crear un informe. Por el contrario, el uso de las mismas solicitudes de datos en la aplicación web subyacente y para la generación de informes elimina el riesgo de posibles incoherencias entre los datos mostrados en la pantalla y el informe impreso.

ReportBro permite diseñar fácil y rápidamente informes con un diseño perfecto. Imprimir la página actual en el pie de página es tan sencillo como insertar variables predefinidas para la página actual y el recuento total de páginas. Incluso para generar salidas Excel no se necesita ninguna configuración adicional debido a que ReportBro tiene una estructura muy clara es fácil de aprender.

2. Libreria reportbro-lib

ReportBro-lib es una librería para generar informes PDF y XLSX escrita en python para su uso conjunto con Django. Las plantillas de informes se pueden crear con ReportBro Designer, un plugin Javascript que se puede integrar en su aplicación web.

3. ReportBro Admin

Proyecto libre que une las librerías JavaScript de ReportBro designer y la libreria reportbro-lib.

Este proyecto constituye una app de django que permite la administración y fácil integración de todas las librerías referentes a ReportBro.
